4. 2진수 문자열(`010` 등)을 입력 받아 10진수와 8진수로 각각 변환한 결과를 출력하는 함수를 작성하세요.

- 문제를 잘 읽어보세요.  
  반환이 아니고 **출력**입니다.  
  그래도 주어진 문제에 대한 해결은 잘 했어요.

```javascript

function change(str) {
    return parseInt(str, 2);
}
change(str);


let str = '1011'
function changes(str) {
   let str10 = parseInt(str, 2);
   return str10.toString(8);
}
changes(str);
```
